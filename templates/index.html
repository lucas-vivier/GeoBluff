Regles du jeu

But : Se debarrasser de toutes ses cartes en premier.

Deroulement :

  • Une categorie est tiree au hasard (Population, Superficie, PIB)
  • Une carte de reference est placee au centre
  • A tour de role, placez une carte sur le plateau
  • Les cartes doivent etre ordonnees de la plus grande valeur (gauche) a la plus petite (droite)

Bluff :

  • Si vous pensez que l'ordre est faux, criez "BLUFF !"
  • Toutes les cartes sont revelees
  • Si l'ordre etait correct : vous recuperez toutes les cartes
  • Si l'ordre etait faux : l'adversaire recupere les cartes

Victoire :

  • Quand vous posez votre derniere carte, nommez sa capitale
  • Bonne reponse = Victoire !
  • Mauvaise reponse = la carte revient dans votre main

Source des donnees : Banque Mondiale 2023
